# DeutschQuest – Deutschprofis A1/A2/B1

Proyecto listo para GitHub Pages.

## Cómo usar
1. Sube toda la carpeta al repositorio.
2. Abre `index.html`.
3. Cada botón abre una lección.
4. Reemplaza los archivos JSON en `data/A1`, `data/A2`, `data/B1` con el vocabulario oficial.

## Formato de cada JSON

```json
{
  "level": "A1",
  "lesson": 1,
  "title": "Deutschprofis A1 - Lektion 1",
  "vocabulary": [
    {"de": "Hallo", "es": "Hola"}
  ],
  "sentences": [
    {"text": "____! Wie geht es dir?", "answer": "Hallo"}
  ]
}
```
